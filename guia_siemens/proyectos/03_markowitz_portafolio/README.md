# Proyecto 03: Portafolio Markowitz

Objetivo: simular retornos de varios activos, generar portafolios aleatorios y encontrar combinaciones con mejor Sharpe ratio y menor volatilidad.

## Que practicas

- Retornos.
- Matriz de covarianza.
- Volatilidad de portafolio.
- Sharpe ratio.
- Frontera eficiente aproximada por simulacion.

## Como correr

Desde `guia_siemens`:

```bash
python proyectos/03_markowitz_portafolio/markowitz_portafolio.py
```
