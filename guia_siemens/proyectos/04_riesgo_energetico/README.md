# Proyecto 04: Riesgo energetico

Objetivo: simular costo energetico bajo incertidumbre de demanda, precio y outages.

## Que practicas

- Monte Carlo aplicado a energia.
- Demanda y precio inciertos.
- Eventos de falla.
- Percentiles de costo.
- Probabilidad de exceder presupuesto.

## Como correr

Desde `guia_siemens`:

```bash
python proyectos/04_riesgo_energetico/riesgo_energetico.py
```
